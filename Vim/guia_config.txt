- instalar vim desde 0
- instalar git
- instalar curl

- agregar la ruta bin de git all PATH del sistema

- crear un cmd llamado curl.cmd con

#####################
@rem Do not use "echo off" to not affect any child calls.
@setlocal

@rem Get the abolute path to the parent directory, which is assumed to be the
@rem Git installation root.
@for /F "delims=" %%I in ("%~dp0..") do @set git_install_root=%%~fI
@set PATH=%git_install_root%\bin;%git_install_root%\mingw\bin;%PATH%

@if not exist "%HOME%" @set HOME=%HOMEDRIVE%%HOMEPATH%
@if not exist "%HOME%" @set HOME=%USERPROFILE%

@curl.exe %*
#####################

- copiarlo en la carpeta bin de git

- clonar vundle en la carpeta vimfiles/bundle/vundle

>> git clone https://github.com/gmarik/vundle.git vimfiles/bundle/vundle

agregar la siguiente conf al _vimrc o usar el personalizado

#############
set nocompatible              " be iMproved, required
filetype off                  " required

" set rtp+='~/vimfiles/bundle/vundle/'
" let path='~/vimfiles/bundle'

set rtp+=$VIM/vimfiles/bundle/vundle/
" let path='$VIM/vimfiles/bundle'
call vundle#rc()

Bundle 'gmarik/vundle'

filetype plugin indent on 
#############

abrir vim y ejecutar :BundleInstall con eso instalara el resto de los paquetes
